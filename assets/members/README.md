# Best Members Images

Place best member photos in this directory.

## Naming Convention
- `member-1.jpg`
- `member-2.jpg`
- `member-3.jpg`
- etc.

## Image Specifications
- Recommended size: 400x400px
- Format: JPG or PNG
- Aspect ratio: 1:1 (square)

Images will be automatically loaded by the BestMembers component based on the `content/best-members.json` configuration.
