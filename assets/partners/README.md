# Partner Logos

Place partner organization logos in this directory.

## Naming Convention
- `partner-1.png`
- `partner-2.png`
- `partner-3.png`
- etc.

## Image Specifications
- Recommended size: 300x150px (2:1 aspect ratio)
- Format: PNG (with transparency preferred)
- Background: Transparent or white

Logos will be automatically loaded by the Partners component based on the `content/partners.json` configuration.
