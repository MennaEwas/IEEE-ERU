# Officers Images

Place officer headshot images in this directory.

## Naming Convention
- `officer-1.jpg`
- `officer-2.jpg`
- `officer-3.jpg`
- etc.

## Image Specifications
- Recommended size: 400x400px
- Format: JPG or PNG
- Aspect ratio: 1:1 (square)

Images will be automatically loaded by the Leadership component based on the `content/officers.json` configuration.
